CreateAndOpenScene(Action<VxScene>, string, bool)

Create a new scene and optionally opens it in Designer. An optional lambda action can be provided to create the initial nodes in this scene.

VxScene CreateAndOpenScene(Action<VxScene> buildSceneAction = null, string nameTemplate = null, bool withLayer3D = true)

Parameters

buildSceneAction Action<VxScene>

If specified, this action is called to build the scene

nameTemplate string

An optional name template, default its "New". The template will also create an increasing number if it already exists ("New1", "New2", ...)

withLayer3D bool

If set to true, the new scene will contain a default 3D layer. Default is true

Returns

VxScene

The created scene
