The class represents a menu entry used in Designer. For example a VxMainMenu provides a list of such menu entries to describe the main menu.

Remarks

A VxMenuEntry has several properties to describe the appearance of the entry. This properties a only read once, while building the menu. Updating or changing the values may not affect the appearance in the UI!
