Interface IVxDataObject

Namespace
Ventuz.Extension
Assembly
Ventuz.Extension.dll

The interface IVxDataObject is a simple bridge to IDataObject extended by some Vx related properties. An implementation of this interface is passed to the Probe and Create methods of a VxDataObjectHandler.
The members of this interface can only be accessed from the main thread!
