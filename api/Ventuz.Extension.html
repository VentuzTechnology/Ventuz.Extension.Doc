VxLink

A VxLink represents the actual position of a hierarchy node in the hierarchy tree. Layer are actual also hierarchy nodes therefor are child layers of a layer also linked to each other. A VxLink points from a VxOutput (FromOutput) owned by a FromNode to a node (ToNode) A hierarchy node can be linked zero, one or multiple times. If a hierarchy is not linked it appears in the "unlinked nodes" area of the Hierarchy Editor in Designer. Unlinked nodes are validated, but never rendered.
